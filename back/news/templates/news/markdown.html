Tutoriel Markddown

Un certain nombre de pages de Upont prennent en charge le markdown pour faire des jolis posts, donc un petit tutoriel s'impose.

Les fonctions de base

  • Les titres: pour créer un titre, il faut encardrer le texte par "#". Plus il y en a, plus le titre est petit (on peut descendre jusqu'à 6).
  • Emphase : pour mettre en italique ou en gras, il faut encadrer respectivement par "*" et "**"
  • Les listes : pour faire une liste ordonnée ou non, il faut utiliser "1. " ou "- " (il faut impérativement un espace après le tag)
  • Les caractères spéciaux : pour ajouter les caractères liés au markdown sans rendre le markdown, il faut ajouter un "/" devant

Les fonctions avancées

En plus de ces fonctions de base, il existe quelques fonctions plus avancées:

  • Les liens hypertextes : pour changer le texte du lien, il faut écrire le texte entre "[" et "]", et le lien entre "(" et ")".
  • Les styles CSS : il est possible d'ajouter des styles CSS pour des blocs (soit des paragraphes séparés par des sauts de ligne, soit des blocs de markdown). La syntaxe n'est pas évidente car cela ne fait pas partie du standard Markdown, donc n'hésitez pas à regarder les exemples. Pour cela, il faut ajouter après le bloc:
    • {: style="color:green";} pour changer la couleur. Vous pouvez remplacer "green" par n'importe quelle couleur hexadécimale.
    • {: style="text-decoration:underline";} pour souligner. Tous les styles de soulignage sont disponibles à cette adresse.
